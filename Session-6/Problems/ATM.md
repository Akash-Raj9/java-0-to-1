# Problem 3: Simple ATM System

### Task:
Write pseudocode for a simple ATM system. The system should:
1. Ask for the user's PIN.
2. Ask for a withdrawal amount.
3. Check if the balance is sufficient.
4. If the balance is sufficient, subtract the amount and display the new balance.
5. If the balance is insufficient, display an error message.

### Instructions:
- Prompt the user for their PIN.
- Prompt the user for a withdrawal amount.
- Check if the balance is enough to cover the withdrawal.
- If yes, subtract from the balance and print the remaining balance.
- If not, print an error message.

### Example:
- PIN: 1234
- Balance: $500
- Withdrawal: $200
- Output: New Balance: $300
